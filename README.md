# Montinho Nutrition Strategy

> Um **Sistema Inteligente de Decisão Nutricional** para uso profissional — não um gerador de dietas.

O **Montinho Nutrition Strategy** existe para responder uma única pergunta:

> "Qual é a melhor estratégia nutricional para **ESTA** pessoa?"

O software organiza o raciocínio clínico e estratégico do profissional (Montinho Personal),
auxiliando na tomada de decisões individualizadas para cada aluno. A dieta é apenas uma
**consequência** das decisões tomadas pelo sistema — nunca o ponto de partida.

O sistema **nunca substitui** o julgamento profissional. Seu papel é **potencializá-lo**.

---

## Princípios Fundamentais

1. Individualização.
2. Baseado em evidências.
3. Alta aderência.
4. Simplicidade quando possível; complexidade apenas quando necessária.
5. Justificar todas as decisões importantes e explicar seus riscos.
6. Priorizar resultados sustentáveis e de longo prazo.
7. Nunca recomendar protocolos extremos sem justificativa técnica.

Consulte o [Documento Mestre](docs/00-vision-product-philosophy.md) para a filosofia completa.

---

## O coração do sistema — Nutrition Decision Engine (NDE)

Toda decisão passa obrigatoriamente pelo NDE, que integra diagnóstico, objetivos, rotina,
preferências, restrições, histórico, psicologia, comportamento alimentar, treinamento e os
bancos inteligentes (alimentos, estratégias, suplementos, casos e evidências) para produzir
uma **estratégia personalizada**.

A hierarquia de decisões é sempre respeitada, nesta ordem:

```
Diagnóstico → Objetivo → Estratégia → Aderência → Macros →
Distribuição das refeições → Escolha dos alimentos →
Estratégias comportamentais → Suplementação → Documento final
```
